# deep_learn_school_deploy


Код для простого чат-бота на python, классифицирующего сообщения на токсичные и не токсичные. 


## Установка

```bash
git clone repo
```

```bash
pip install requiremets.txt
```

```bash
python app.py
```

## Замечание
Перед установкой следует указать `token` в файл `config.py`
